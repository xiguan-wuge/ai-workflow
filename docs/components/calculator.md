# Calculator 计算器

支持基本算术运算的简单计算器组件。

## 示例

```vue
<template>
  <Calculator />
</template>

<script setup>
import { Calculator } from '@ai-workflow/components'
</script>
```

## 功能

- 基本运算：加、减、乘、除
- 清零功能
- 小数支持
- 响应式设计

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| - | - | - | 无需传入任何 props |
