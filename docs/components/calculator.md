# Calculator 计算器

支持基本算术运算的简单计算器组件。

## 示例

:::demo
calculator/basic
:::

## 功能

- 基本运算：加、减、乘、除
- 清零功能
- 小数支持
- 百分比、退格、正负号切换
- 自适应宽度（`min(340px, 100%)`）
- 键盘支持：`0-9`、`.`、`+ - * /`、`Enter/=`、`Backspace`、`Esc`、`%`、`F9(±)`

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| - | - | - | 无需传入任何 props |
