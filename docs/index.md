# AI Workflow 文档

Vue 3 组件库，专为 AI Workflow 项目打造。

## 特性

- 🎨 精美的 UI 组件
- 📦 易于安装和使用
- 📚 完整的文档支持
- 🧪 交互式 Playground

## 快速开始

```bash
pnpm add @ai-workflow/components
```

```vue
<script setup>
import { Calculator } from '@ai-workflow/components'
</script>

<template>
  <Calculator />
</template>
```

## 组件列表

- [Calculator](/components/calculator) - 简单计算器组件
- [HelloWorld](/components/helloworld) - 基础示例组件
