# 快速开始

## 安装

```bash
pnpm add @ai-workflow/components
```

## 使用

```vue
<script setup>
import { Calculator, HelloWorld } from '@ai-workflow/components'
</script>

<template>
  <div>
    <Calculator />
    <HelloWorld msg="欢迎！" />
  </div>
</template>
```

## Playground

在 Playground 中体验组件（详见 `playground/` 目录）。
