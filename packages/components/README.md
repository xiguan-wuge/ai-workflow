# @ai-workflow/components

Vue 3 Component Library for AI Workflow projects.

## Installation

```bash
pnpm add @ai-workflow/components
```

## Usage

```vue
<script setup>
import { Calculator, HelloWorld } from '@ai-workflow/components'
</script>

<template>
  <Calculator />
  <HelloWorld msg="Welcome!" />
</template>
```

## Components

### Calculator
A simple calculator component with basic arithmetic operations.

### HelloWorld
A basic Hello World component for demonstration.
